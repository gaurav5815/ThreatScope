# ThreatScope – Threat Intelligence Dashboard
ThreatScope is a real-time cybersecurity threat intelligence dashboard that collects, analyzes, and visualizes cyber threat data.

## Features
- Real-time threat intelligence collection
- MongoDB database storage
- Flask backend API
- React frontend dashboard
- Threat monitoring and analysis

## Tech Stack
- Frontend: React
- Backend: Python Flask
- Database: MongoDB
